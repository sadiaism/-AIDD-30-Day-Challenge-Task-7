## ✔ Short answers for each of the 5 concepts

1️⃣ /constitution
Constitution aik project ka basic aur permanent rules ka set hota h jo complete project pr apply hota h aur change nh hota, ye global standards deta h jissy har feature ya task follow krta h.

2️⃣ /specify
Specify phase my ap apny vague idea ko clear aur testable requirements my convert krty hain,jahan success criteria define kiye jaty hain.ye AI aur project ke liye exact direction provide krta h. 

3️⃣ /plan
Plan phase my specification ko structured plan my convert kiya jata h,jismy work phases,dependencies aur import design decision (ADRs) document hote hain.

4️⃣ /tasks
Tasks small,independent work units hoty hain ,jinka aik clear acceptance criteria hota h.Checkpoints my human har phase ko review aur approve krta h,jis sy workflow pr control rhta h aur errors jldi pkry jaskty hn.

5️⃣ /implement
Implementation = AI ke sath tasks execute karna + output ko spec ke against validate karna, checkpoints aur iteration loops use karke quality aur accuracy ensure karna.

Implementation jo h wo AI ky sath task ko execute krna aur plus output ko spec ky against validate krna ,checkpoints aur iteration loops use krky quality aur accuracy ensure krna.
